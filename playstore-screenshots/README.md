# 구글 플레이 스토어 스크린샷 준비

## 폴더 구조
```
playstore-screenshots/
├── raw/          ← 바탕화면에서 가져온 원본 이미지
├── phone/        ← 휴대폰 규격으로 리사이즈된 최종본 (1080x1920)
└── README.md
```

## 구글 플레이 스토어 스크린샷 규격 (휴대폰)
- **비율:** 16:9 또는 9:16
- **권장 크기:** 1080 x 1920 px (세로) 또는 1920 x 1080 px (가로)
- **프로모션 이용 시:** 각 변이 **최소 1,080px** 이상인 스크린샷 **4개 이상** 필요
- **최대:** 긴 변 3840px 이하
- **형식:** PNG 또는 JPEG (알파 없음)
- **개수:** 최소 2장, 최대 8장 (프로모션용은 4장 이상)

## 사용 방법 (나중에)

1. 바탕화면에 저장해 둔 스크린샷을 `raw/` 폴더로 복사
2. 아래 스크립트 실행:
   ```bash
   python resize-for-playstore.py
   ```
3. `phone/` 폴더의 최종 이미지를 플레이 콘솔에 업로드

## 7인치 태블릿 스크린샷

바탕화면에 `holic1.png`, `holic2.png`, `holic3.png`, `holic4.png` 를 두고 실행:

```bash
python resize-tablet7.py
```

→ 바탕화면에 `holic1-tablet7.png`, `holic2-tablet7.png` … 형태로 저장 (1200×1920)
